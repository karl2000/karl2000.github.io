一个简单的入门试验

first_drv.c源程序：

first_drv_test.c程序

在Ubuntu上执行
sudo make ARCH=arm CROSS_COMPILE=arm-linux-gnueabi-
生成first_drv.ko，再执行
sudo arm-linux-gnueabi-gcc first_drv_test.c -o first_drv_test
生成first_drv_test

拷贝上述first_drv.ko和生成first_drv_test到树莓派上

sudo insmod first_drv.ko
sudo mknod /dev/xxx c 111 0
sudo chmod 755 first_drv_test (设置执行权限)
sudo ./first_drv_test

打印信息通过dmesg查看，或者用下面命令在另一个终端查看
tail -f /var/log/syslog

卸载用
sudo rmmod first_drv.ko

Hello World example

Classic Hello World example for a loadable kernel module (LKM).

mkdir hello && cd hello
Makefile

obj-m := hello.o
hello.c

#include

#include

int hello_init(void)
{
pr_alert(“Hello World :)\n”);
return 0;
}
void hello_exit(void)
{
pr_alert(“Goodbye World!\n”);
}
module_init(hello_init);
module_exit(hello_exit);
$ make -C /lib/modules/$(uname -r)/build M=$(pwd) modules
$ sudo insmod hello.ko
$ dmesg | tail -1
[58728.008906] Hello World :)
$ sudo rmmod hello
$ dmesg | tail -1
[58732.440677] Goodbye World!

//这里没执行

Install module (/lib/modules//extra/hello.ko)

$ sudo make -C /lib/modules/$(uname -r)/build M=$(pwd) modules_install
$ sudo depmod
$ sudo modprobe hello
